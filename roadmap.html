内部合作方
六大合作团队
跨团队协作是三个 Solution 落地的关键——每个团队有明确的职责边界和介入时机。
合作方 01
灵犀 / 一方数据产品
提供 UGC 数据基础设施，支撑 Brand Imagery 诊断、品牌心智实时追踪、一方销售归因全链路数据能力。
Solution A 核心 Solution C 诊断
合作方 02
AI 产品团队
构建 Brand Attributes 自动打标模型、UGC 情感分析、归因模型——让 Kantar 级别的调研能力以 AI 速度实时产出。
Solution A MVP 产品化开发
合作方 03
营销实验室
负责 Brand Attributes 行业框架定义（Solution A）；主导 Red Brand Studio 品牌招募与共创流程设计（Solution C）。
Attribute 定义 品牌招募
合作方 04
社区团队
主导央视/卫视 IP 资源联合谈判，建立框架协议；协调 UGC 话题机制，打通 IP 内容到社区二次传播的通路。
Solution B 核心 央/卫视谈判
合作方 05
IP 营销中心
策划 Solution B 定制 IP 联合解决方案——IP 筛选、玩法设计、品牌招商 Deck、落地执行全程主导。
IP 方案策划 品牌招商
合作方 06
渠道团队
主导 Red Partners 外部品牌咨询/设计公司招标，建立合作伙伴体系，确保 Solution C 有高质量外部交付能力支撑。
Red Partners 招标 Solution C
